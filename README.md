## Project Scope
The project focuses on:
- Verifying datasets of notable people and riding-level facts
- Highlighting selected examples through map-based storytelling
- Providing an interactive dashboard for exploring the full dataset

The verified datasets will be incorporated into the **Riding Profiles**, a widely used online resource of the Ontario Legislative Library.

## Data
This repository includes two tabular datasets:
- **[Notable people](data/notable-people.xlsx)**
- **[Fun facts](data/fun-facts.xlsx)**

### Data Sources
- Riding facts were gathered by librarians at the **Ontario Legislative Library**
- Data on notable people were extracted from a public Wikipedia dataset compiled by **[Laouenan et al. (2022)](https://www.nature.com/articles/s41597-022-01369-4)**
- Short biographies were extracted and edited from Wikipedia
